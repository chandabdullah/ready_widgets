## Installation

Add `ready_widgets:` to your **pubspec.yaml** dependencies then run `flutter pub get`

```
 dependencies:
  ready_widgets:
```

## Import

Add this line to import the package.
```
import 'package:ready_widgets/ready_widgets.dart';
```


## How to use

```dart
CustomInput(
    title: 'Name',
    hint: 'Enter your name',
    controller: controller,
);
```
